# AutoEmbed templates

Ce dossier contient des templates recuperes (notamment WordPress) qui peuvent servir de base UI/UX.

## API AutoEmbed (rappel)

### Film

```text
https://player.autoembed.cc/embed/movie/{id}
```

### Episode TV

```text
https://player.autoembed.cc/embed/tv/{id}/{season}/{episode}
```

### Parametre serveur

```text
?server=2
```
